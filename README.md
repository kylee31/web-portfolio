# 👩‍💻 웹 포트폴리오
React를 사용하여 TypeScript로 구현한 웹 포트폴리오 (2023)
<br/><br/>
## 구현 및 사용 기술(특징)
- TypeScript 적용한 React 프로젝트 (firebase 배포).
- SPA 형식으로 반응형 웹 페이지 구현.
- 레이아웃 컴포넌트를 만들어 카테고리(페이지) 정리.
